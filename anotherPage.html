Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML Returns the HTML content inside an element. , InnerText Returns only the visible text inside an element. , TextContent Similar to innerText, but it gets all text, including hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where you add a single event listener to a parent element instead of multiple event listeners on individual child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the Event Propagation model where an event starts from the target element and bubbles up through its parent elements until it reaches the document object.
